# CAC & LTV Calculator

A static web app that calculates Customer Acquisition Cost (CAC), Customer Lifetime
Value (LTV), and the LTV:CAC ratio for a business.

This is a pre-built static site (Next.js static export) — no build step required.

## Deploy on Render (Static Site)
1. Upload all of these files to a GitHub repo (keep the folder structure: `index.html`,
   `404.html`, and the `_next/` folder at the top level).
2. In Render: New + -> Static Site -> connect the repo.
3. Build Command: leave EMPTY.   Publish Directory: `.`
4. Create. Done.

## Or deploy in one step (no GitHub needed)
Drag this folder onto https://app.netlify.com/drop and it goes live instantly.
